project proposal

Functionality

  1. All the content on the home page is viewable to every user , irrespective of whether he is logged in or not
  2. Any user can can play a song and see its details
  3. Using spotify API, user can search song tracks using the "Search" tab
  4. Any user can search a track and listen to it from the details page
  5. The user has to login/register to enter any data into the system
  6. To register
    1. All the fields are mandatory
    2. The user must provide a valid email id(valid format)
    3. The user must choose a non existing username
    4. The passwords entered must match
    5. Upon clicking register button, user will be asked to enter above fields, if any
  7. To Login
    1. The user must enter valid credentials else the user will be prompted to enter valid credentials
  8. Any logged in user can follow any other user
  9. All follower and following users can be viewed in the profile page
  10. A logged in user can edit only his profile
  11. Only the logged in user can like or unlike
  12. Only logged in user can create a new comment or delete his comment
  13. Only logged in user can view all his likes in his profile page
